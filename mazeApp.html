Description du Projet

MazeApp est une application Java mêlant génération de labyrinthes, interfaces graphiques et logique de jeu. Le joueur peut explorer différents types de labyrinthes (aléatoires, parfaits, division récursive), chacun généré par un algorithme spécifique avec différent types de vues (total, local, exploration).

Le projet mobilise des compétences en Joo, JavaFX, modélisation MVC, gestion de scènes, algorithmes de parcours, et tests unitaires. Nous avons également implémenté un système de sauvegarde, un mode progression, des paramètres personnalisés (taille, difficulté, distance minimale), ainsi qu’une interface utilisateur complète et modulable.

L’objectif était de produire une application fonctionnelle, claire et évolutive, mêlant logique algorithmique et expérience utilisateur.
